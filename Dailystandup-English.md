# Daily Stand-up Script Guide

## Versi Sempurna

```
You: "Morning team! Thanks for being here. Let's begin our daily stand-up. Andi, you can start."
Andi: [shares update]
You: "Thanks Andi. Sinta, you're next."
Sinta: [shares update]
You: "Thanks Sinta. Last one, Rama, please go ahead."
Rama: [shares update]
You: "Perfect! That's everyone. Thanks team, let's get back to work!"
```
